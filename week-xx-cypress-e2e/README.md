# Cypress End-to-End Testing Exercise

## 1. Giới thiệu
Dự án này sử dụng **Cypress** để thực hành kiểm thử tự động End-to-End (E2E) cho trang web mẫu  
**https://www.saucedemo.com**.

Các kịch bản kiểm thử bao gồm:
- Đăng nhập thành công
- Đăng nhập thất bại
- Thêm sản phẩm vào giỏ hàng
- Sắp xếp sản phẩm theo giá (low → high)
- Xóa sản phẩm khỏi giỏ hàng
- Quy trình thanh toán (Checkout)

---

## 2. Yêu cầu môi trường
- Node.js **>= 14**
- npm
- Visual Studio Code (hoặc IDE bất kỳ)
- Trình duyệt Chrome / Electron
